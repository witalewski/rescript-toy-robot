#Rescript Toy Robot

[Coding excercise](https://github.com/tartansandal/mr-robot/blob/master/brief.md) loosely based on [a book](https://leanpub.com/elixir-toyrobot) by Ryan Brigg.


## Installation

```sh
npm install
```

## Build

- Build: `npm run build`
- Clean: `npm run clean`
- Build & watch: `npm run watch`

## Run

`npm run start`
